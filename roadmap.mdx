### Roadmap

The following features are planned for JS-Confuser:

- VM Obfuscation: [JS-Confuser-VM](https://github.com/MichaelXF/js-confuser-vm) has started!
- Webpack & Build Plugins
- Babel Rewrite
- Custom String Concealing functions
- Custom Lock code / Dead code insertions
- Improved Code Transforms
- Improved API Interface

---

A '2.0 version' of JS-Confuser is currently in the works. Please feel free to suggest new features on the [GitHub issues page](https://github.com/MichaelXF/js-confuser).
