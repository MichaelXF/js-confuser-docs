### Input / Output

This example showcases how `Hexadecimal Numbers` transforms the code. Try it out by changing the input code and see changes apply in real-time.

```js title="Input.js" lines interactive-mode="obfuscate"
// Input.js
var ten = 10;
var negativeSixteen = -16;
var float = 0.01;

console.log(ten, negativeSixteen, float);

// Output.js
var c4wpKB4 = 0xa,
  P9psmK = -0x10,
  yhDkmbU = 0.01;
console.log(c4wpKB4, P9psmK, yhDkmbU);
```

---





### Usage Example

The provided code example will obfuscate the file `input.js` and write the output to a file named `output.js`.

```js title="Usage Example" lines
import JSConfuser from "js-confuser";
import {readFileSync, writeFileSync} from "fs";

// Read input code
const sourceCode = readFileSync("input.js", "utf8");
const options = {
  target: 'browser',
  hexadecimalNumbers: true,
  renameVariables: false,
};

JSConfuser.obfuscate(sourceCode, options).then((result)=>{
  // Write output code
  writeFileSync("output.js", result.code);
}).catch(err=>{
  // Error occurred
  console.error(err);  
});
```
