#### Input / Output

This example showcases how `Duplicate Literals Removal` transforms the code. Try it out by changing the input code and see changes apply in real-time.

```js title="Input.js" lines interactive-mode="obfuscate"
// Input.js
var myBool1 = true;
var myBool2 = true;

console.log(myBool1 === myBool2); // true

// Output.js
const NRYcms = [!0];
var OT0bif = NRYcms[0],
  ulHLOgo = NRYcms[0];
console.log(OT0bif === ulHLOgo);
```

#### Usage Example

The provided code example will obfuscate the file `input.js` and write the output to a file named `output.js`.

```js title="Usage Example" lines
import JSConfuser from "js-confuser";
import {readFileSync, writeFileSync} from "fs";

// Read input code
const sourceCode = readFileSync("input.js", "utf8");
const options = {
  target: 'browser',
  duplicateLiteralsRemoval: true,
};

JSConfuser.obfuscate(sourceCode, options).then((result)=>{
  // Write output code
  writeFileSync("output.js", result.code);
}).catch(err=>{
  // Error occurred
  console.error(err);  
});
```
